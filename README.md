# Car Sales Analysis Dashboard

## Project Overview

This project presents an exploratory analysis of automobile sales data using Microsoft Excel. The dataset contains information about vehicle manufacturers, models, sales performance, pricing, resale values, engine specifications, fuel efficiency, and other automotive characteristics.

The objective is to identify trends in vehicle sales, compare manufacturer performance, analyze pricing strategies, and understand relationships between vehicle specifications and market success.

---

## Dataset Information

The dataset contains 155 vehicle records and 19 attributes.

### Key Features

- Manufacturer
- Model
- Unit Sales
- Price
- Year Resale Value
- Retention Percentage
- Retention Value
- Engine Size
- Horsepower
- HP Level
- Vehicle Type
- Fuel Efficiency
- Power Performance Factor
- Wheelbase
- Width
- Length
- Curb Weight
- Fuel Capacity
- Latest Launch Date

---

## Objectives

- Analyze vehicle sales performance.
- Compare manufacturers based on sales volume.
- Study the impact of pricing on sales.
- Examine resale value retention.
- Evaluate relationships between horsepower, engine size, and performance.
- Visualize vehicle characteristics through interactive charts.

---

## Dashboard Components

### Bar Charts
- Sales comparison among manufacturers.
- Performance comparison of vehicle models.

### Column Charts
- Unit sales distribution.
- Vehicle type analysis.

### Pie Charts
- Market share by manufacturer.
- Vehicle category distribution.

### Sunburst Charts
- Hierarchical view of manufacturers and models.

### Dashboard
An integrated dashboard combining multiple visualizations for business insights and decision-making.

---

## Key Insights

- Some manufacturers dominate total unit sales.
- Higher-priced vehicles generally show stronger resale values.
- Engine size and horsepower positively influence performance metrics.
- Fuel efficiency varies significantly across vehicle categories.
- Retention percentage provides insight into long-term vehicle value.
